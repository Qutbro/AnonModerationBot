# 🤖 Telegram Bot для анонимной модерации контента

Бот для Telegram, который принимает сообщения от пользователей и отправляет их администраторам на модерацию. После одобрения контент публикуется в канал.

---

## 🚀 Возможности

- 📩 Приём сообщений любого типа:
  - Текст
  - Фото
  - Видео
  - Документы
  - Голосовые сообщения
  - Кружки (video_note)
- 👮 Система модерации через inline-кнопки
- ✅ Одобрение и публикация в канал
- ❌ Отклонение сообщений
- 🔇 Мут пользователя на 15 минут
- ⚠️ Выдача предупреждения
- 🚫 Полная блокировка пользователя
- 📂 Хранение базы сообщений в `send.bd`
- 🛡 Поддержка обычных админов и супер-админов

---

## 🛠 Используемые технологии

- Python 3.10+
- python-telegram-bot
- JSON (для хранения данных)
- asyncio
- InlineKeyboard (Telegram API)

## 🔐 Безопасность

❗ Никогда не публикуйте токен бота в открытом репозитории.

Вместо:

```python
BOT_TOKEN = "your_token_here"
```

Используйте переменные окружения:

```python
import os
BOT_TOKEN = os.getenv("BOT_TOKEN")
```

Запуск:

```bash
export BOT_TOKEN=your_token_here
python bot.py
```

## 👮 Система ролей

### 🔹 Администраторы (`ADMIN_IDS`)
Получают сообщения на проверку и могут:

- Одобрить
- Отклонить
- Замутить пользователя
- Предупредить
- Заблокировать

### 🔹 Супер-администраторы (`SUPER_ADMINS`)
Дополнительно могут:

- Открывать админ-панель
- Выгружать базу `send.bd`

Команда:

```
/admin
```

---

## 📌 Как работает бот

### 1️⃣ Пользователь отправляет сообщение боту

Бот:
- Проверяет блокировку
- Проверяет мут
- Сохраняет сообщение в `send.bd`
- Отправляет его всем администраторам

---

### 2️⃣ Администратор получает сообщение с кнопками

Кнопки:

- ✅ Одобрить
- ❌ Отклонить
- 🔇 Замутить 15 мин
- ⚠️ Предупреждение
- 🚫 Заблокировать

---

### 3️⃣ Если сообщение одобрено

Бот публикует его в канал (`CHANNEL_ID`) и добавляет ссылку:

```
✉️ Отправить своё сообщение анонимно
```
